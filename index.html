Key Concepts
Intra-state
GST = CGST + SGST (each = GST/2)
Inter-state
GST = IGST only
Tax Liability
= Output Tax − Input Tax
Consumer pays
= SP + CGST + SGST (or IGST)
GST Formulas
CGST = SGST
= (GST Rate / 2) % of SP
IGST
= GST Rate % of SP
Net Tax to Govt
= Output Tax − Input Tax
SP with profit
= CP × (1 + Profit%/100)
SP with discount
= MP × (1 − Discount%/100)

Key Formulas
Interest
I = P × [n(n+1)/(2×12)] × r/100
Maturity Value
MV = P × n + I
Banking Formulas
Interest on RD
I = P × [n(n+1)/(2×12)] × r/100
Maturity Value
MV = (P × n) + I
Find Rate r
r = (I × 100 × 12) ÷ [P × n(n+1)/2]
Find Monthly P
Equate P×n + I = MV, solve for P
Find Period n
Form quadratic in n, reject negative root

= (Total FV × Rate%) ÷ 100

Shares & Dividend Exam Tips
⚠️ Dividend is always on Face Value, NEVER on Market Value!
⚠️ Return % is always on Market Value (actual investment).
✅ Comparing investments: use same base investment in both cases.
✅ Change-in-income: calculate old income first, then new income after reinvestment.
✅ Premium → MV = FV + Premium. Discount → MV = FV − Discount.
✅ 4 steps: Find FV total → Dividend → Investment → Return %
